YouTube Transcript Exporter
===========================

Description
-----------
This is a desktop application designed to extract transcripts (defaulting to English) from any public YouTube video.

Use Cases
---------
The app aims to support:

*   **English Learners**: Facilitates shadowing and language practice.
*   **Text Extraction**: Designed for users who need seamless text extraction and editing capabilities based on transcripts directly sourced from YouTube.

Features
--------
*   **Always on Top**: Includes an "Always on top" feature to enhance convenience, removing the need for users to switch back and forth between screens.

Privacy Policy
--------------
Regarding security mechanisms, the app commits to **not collecting any user information**. It only uses the network to query data directly from YouTube.

Limitations & Warnings
----------------------
*   **Direct Query Warning**: As queries are made directly to YouTube's source data, spamming requests may result in YouTube blocking your public IP address. **Warning: Do not spam continuously; you will be the one disadvantaged.**
*   **Platform Support**: Due to the author's limited resources, the application currently only supports desktop environments on **macOS Silicon**.
